# Desafio técnico - FinPec

### Instalação - Backend

```bash
cd backend
yarn
```

### Gerar Migration

```bash
yarn typeorm migration:generate -n InitialMigrate
```

### Rodar a Migration

```bash
yarn typeorm migration:run
```

### Rodar o backend

```bash
cd backend
yarn start:dev
```

### Instalação - Frontend

```bash
cd frontend
yarn
```

### Rodar frontend

```bash
cd frontend
yarn dev
```
